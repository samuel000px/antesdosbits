Como funcionam os cartões FORTRAN

FORTRAN transformou fórmulas matemáticas em programas. Mas na época, cada comando precisava estar em um cartão físico. O computador lia a pilha de cima para baixo, obedecendo rótulos, desvios e laços.

READ Coloca valores de entrada nas variáveis, como N, A, B ou TEMP.
IF Compara uma variável e decide se o fluxo deve pular para outro cartão.
DO Repete cartões para somar vários valores ou testar uma sequência.
PRINT Envia o resultado para a saída. Se imprimir cedo demais, perde a missão.
