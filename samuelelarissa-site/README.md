# 7 meses

Site comemorativo de 7 meses de namoro para Samuel e Larissa.

## Como editar

- Abra `index.html` no navegador para ver o site.
- Troque os textos diretamente nos arquivos `.html`.
- Coloque as fotos em `assets/fotos/`.
- Para usar a imagem da capa, salve uma foto como `assets/fotos/capa.jpg`.
- Para o mural da Larissa, salve uma foto como `assets/fotos/larissa.jpg`.
- Para videos, use `assets/videos/video-01.mp4` e `assets/videos/video-02.mp4`.
- Para a aba de musicas, os MP3 ficam em `musica/`.
- Musicas ja adicionadas: `jorge-mateus-seu-astral.mp3`, `gustavo-mioto-last-goodbye.mp3`, `luan-santana-amar-nao-e-pecado.mp3`, `luan-santana-o-amor-coloriu.mp3`, `luan-santana-te-esperando.mp3` e `luan-santana-te-vivo.mp3`.
- Para publicar com dominio, use GitHub Pages, Netlify, Vercel ou outro servico de hospedagem estatica.

## Paginas

- `index.html`: inicio e contador.
- `site-com-musica.html`: modo de navegacao com musica continua.
- `historia.html`: linha do tempo.
- `texto-samuel.html`: pagina final para o texto escrito por Samuel.
- `galeria.html`: album de fotos.
- `musicas.html`: seletor de musicas em MP3.
- `larissa.html`: mural dedicado a Larissa.
- `cartas.html`: mensagens surpresa.
- `quiz.html`: joguinho do casal.
- `futuro.html`: planos e promessas.
