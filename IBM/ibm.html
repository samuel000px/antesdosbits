Tutorial rapido

Como jogar a sala IBM

Monte uma receita de maquinas. O baralho entra bruto, passa pelas maquinas na ordem que voce escolheu e precisa sair como o relatorio esperado.

1. Leia a ordem Veja qual campo precisa virar relatorio: salario por departamento, compra por produto ou venda por regiao.
2. Prepare o baralho Use verificadora para remover erro, reprodutora para criar campos calculados e coladora quando precisar do arquivo mestre.
3. Ordene antes de agrupar A tabuladora so funciona direito se a classificadora ordenar pelo mesmo campo usado em "por".
4. Tabule e imprima A tabuladora gera o relatorio. A impressora apenas entrega o resultado final para comparacao.

Receitas para entender as ordens

Folha de pagamento Verificadora -> Classificadora por dept -> Tabuladora sum salary por dept -> Impressora
Estoque IBM Reprodutora shortage -> Selecionadora shortage > 0 -> Reprodutora orderValue -> Classificadora por sku -> Tabuladora sum orderValue por sku -> Impressora
Vendas por regiao Classificadora por customer -> Coladora customer -> Selecionadora segment = IND -> Classificadora por region -> Tabuladora sum amount por region -> Impressora
